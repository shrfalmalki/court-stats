# نظام إدارة المستفيدين (Beneficiaries Management System)

نظام لإدارة بيانات المستفيدين وتتبع السجلات اليومية للدوالئر المختلفة.

## المميزات

* تسجيل الدخول بنظام أدوار (مدير / موظف).
* إدخال بيانات المستفيدين (الاسم، الهوية، القضية، الدائرة، الصفة، السبب).
* لوحة تحكم للمدير لإدارة الموظفين، الدوائر، الصفات، والأسباب.
* رسوم بيانية وإحصائيات.
* إمكانية تصدير واستيراد البيانات (Excel).

## التثبيت والتشغيل

1. تأكد من تثبيت Node.js.
2. قم بتثبيت المكتبات المطلوبة:
   ```bash
   npm install
   ```
3. تشغيل الخادم:
   ```bash
   npm start
   ```
4. افتح المتصفح على الرابط: `http://localhost:3000`

## بيانات الدخول الافتراضية

* **المدير:**
  * الاسم: `admin`
  * كلمة المرور: `1234`

* **الموظف (مثال):**
  * الاسم: `entry` أو أي اسم موظف مضاف
  * كلمة المرور: `1234`
